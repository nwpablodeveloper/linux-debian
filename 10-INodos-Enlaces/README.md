## INODOS Y ENLACES

### INODO
Se asigna un número de INODO a cada fichero creado de forma automatica
Un INODO tiene la inforamción de METADATOS del fichero
* Tamaño
* Dirección de datos
* Usuario
* Grupo
* etc.
